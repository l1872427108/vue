# vue的优化方式

  1. v-if 和 v-show
  2. 使用 Object.freeze() 方式冻结，阻止数据劫持。
  3. 组件销毁的时候会断开所有与实例联系， 除了addEventListener，需要手动删除。
  4. 图片懒加载
  5. 路由懒加载。
  6. 减少重新渲染和创建dom节点的时间， 虚拟dom。

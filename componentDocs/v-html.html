    # v-html

    v-html 可以插入一段字符串在标签中。

    # 例

    <template>
    <div>
        <p v-html="txt"></p>
    </div>
</template>
<script>
    data() {
        return {
            txt: '<div>1234</div>'
        }
    }
</script>

    可以渲染为一个标签。

    # 注意

    1. v-html 渲染中的子标签将会被覆盖.

    2. 我们要引入自己信任的字符，一旦引入未知的或者带有风险的，将会带来 XSS 攻击。

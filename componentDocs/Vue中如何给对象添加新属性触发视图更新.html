    # Vue中给对象添加新属性触发视图更新。

    Vue 不允许在已经创建的实例上动态添加新的响应式属性。

    1. Vue.set()

    2. Object.assign()

    创建一个新的对象，合并原来对象和混入对象的属性

    this.someObject = Object.assign({}, this.someObject, {newProperty1: 1....})

    1. forceUpdate 可以在 Vue 中做一次强制更新。

    使用 $forceUpdate 迫使 Vue 实例重新渲染。 （影响实例本身和插入插槽内容的子组件）

    # 总结

    少量属性可以使用 Vue.set()

    大量的新属性，可以使用 Object.assign() , 创建一个新对象

    如果真的找不到哪里有错，可以使用 $forceUpdate 强制刷新。
